Instalaciones y Librerías Necesarias

1. Instalar Flask

    pip install Flask

2. Instalar psycopg2

    pip install psycopg2

3. Instalar psycopg2-binary

    pip install psycopg2-binary

4. Verificar instalación de PostgreSQL

    psql --version

5. Crear entorno virtual

    python -m venv venv

Activarlo en Windows

    venv\Scripts\activate

6. Ver versión de Python

    python --version
